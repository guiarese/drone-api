# Drone-api

Desenvolvimento de Drones, Back-End para composição do trabalho de INTEGRATIONS & DEVELOPMENT TOOLS.<p>

Back End: <p>
  - [x] Node JS <p>

## Requirementos
```
node, nodemon, .net, docker
```

## Instalação
```
npm install
```
## Run 
```
npm start 
```
## Deploy Docker 
```
docker-compose up
```
